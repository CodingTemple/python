Data Analytics

Additional Prework for Data Analytics

Get some practice with key technologies used in data analytics.

Exercise 1

Create a Google Sheet to keep track of grades. Use as much creativity as possible. Provide the average of your sample of fake grades. Submit to GitHub when finished.

Exercise 2

  • MySQL Community Server Download and install
  • MySQL Workbench Download and install. This will function as the GUI for MySQL.
  • Create a database using MySQL Workbench using an SQL query.
  • Add a table to the database. Make sure it has at least two different columns! (Price, item, etc.)
  • Insert data into the databse that you created!
  • Run a select query. You can select anything or everything.
  • Save these queries together in SQL file format and upload to GitHub.
